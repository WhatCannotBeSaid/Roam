# Roam Research Deep Optimized Theme

这是一个深度优化的 Roam Research 自用主题，专注于提供沉浸、护眼且高效的写作体验。本主题基于 CSS 变量构建，支持原生深色模式切换，并融合了中国传统配色美学。

## ✨ 主要特性

*   **🎨 护眼配色方案**
    *   **日间模式**：以米黄（`#F5F0E6`）为底，搭配墨青（`#2A5A6A`）主色，采用低饱和度的中国传统色，阅读舒适。
    *   **夜间模式**：适配 `roam-native-dark`，使用深灰（`#2A2A2A`）背景与柔和的文字颜色，降低视觉疲劳。
*   **✒️ 精致排版**
    *   精心挑选的字体栈：标题使用 Futura / 颜真卿楷书，正文使用 Optima / 空山楷，代码使用 Fira Code。
    *   优化的标题层级与间距。
*   **🛠 功能增强**
    *   **沉浸式代码块**：无边框设计，透明背景，专注于代码内容。
    *   **增强复选框**：完成任务后自动添加优雅的删除线样式，且复选框自动隐藏，减少视觉干扰。
    *   **智能搜索框**：集成了同步状态指示灯（同步中显示橙色光晕，完成显示绿色），支持动态展开。
    *   **标签美化**：为 `#Tweet`, `#Book`, `#Idea` 等特定标签定制了专属样式。
    *   **看板优化**：更加现代和紧凑的 Kanban 样式。
*   **🧹 界面净化**
    *   隐藏冗余的帮助按钮、Intercom 图标等干扰元素。
    *   侧边栏与顶栏的去边框处理，更加简洁。

## 🚀 使用指南

### 安装方法

1.  在 Roam Research 中创建一个名为 `roam/css` 的页面。
2.  在页面中创建一个代码块（输入 `/code` 并选择 `CSS`）。
3.  将 [roam.css](roam.css) 文件中的所有内容复制并粘贴到该代码块中。
4.  将代码块下方的语言下拉菜单设置为 `css`。

### 自动日夜模式切换

本主题完美适配 **Roam Native Dark** 插件。推荐安装以下浏览器扩展以实现跟随系统的自动深色模式切换：

*   GitHub 项目: [dive2Pro/roam-native-dark](https://github.com/dive2Pro/roam-native-dark)
